    Submit to Community
    
  

  

    The Vigil community catalog lives on GitHub at
    SusquehannaSyntax/Vigil-Approved-Scripts.
    Submitting opens GitHub's new-file editor with your YAML pre-filled — GitHub will offer to fork the repo and open a PR.
  

  

    How submission works:
    
    
      
  1. Click Open GitHub PR. The file will be created at task.yaml under tasks/.
    2. 
      
  2. Sign in to GitHub if prompted. If you don't have write access, GitHub will automatically fork the repo into your account.
    3. 
      
  3. Click Propose new file on GitHub, then Create pull request.
    4. 
      
  4. A SQSY maintainer reviews and merges. Once merged, every Vigil instance can browse and fork the entry.
    5. 
    

  

  

    Auto-added to your YAML if you haven't set them: author (your Vigil username) and created (today's date). Required by the catalog so every entry is attributed.
